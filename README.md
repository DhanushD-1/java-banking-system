
# Java Banking Management System

A console-based banking system developed in Java that allows users to perform basic banking operations.

## Features

- Create bank account
- Deposit money
- Withdraw money
- Check account balance
- Menu-driven interface

## Technologies Used

- Java
- Object Oriented Programming
- Exception handling
- Scanner class for input

## Concepts Demonstrated

- Class and Object design
- Encapsulation
- Conditional logic
- User interaction through console

## How to Run

1. Compile the program
javac BankSystem.java

2. Run the program
java BankSystem
